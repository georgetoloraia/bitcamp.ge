рЃЋрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃЌ HTML - рЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃљрЃА рЃЊрЃљ рЃљрЃЏ рЃЌрЃљрЃЋрЃерЃў рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА. 

## рЃарЃљ рЃљрЃарЃўрЃА HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў?

HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃАрЃљрЃерЃћрЃюрЃў рЃЏрЃљрЃАрЃљрЃџрЃљрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. рЃЏрЃљрЃЌрЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃћрЃЉрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃбрЃћрЃњрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃЎрЃў рЃЌрЃљрЃЋрЃАрЃЊрЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў. рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃБрЃњрЃћрЃЉрЃарЃљрЃЊ рЃњрЃћрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃюрЃљрЃњрЃарЃФрЃћ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃюрЃљрЃЌрЃћрЃџрЃў рЃњрЃљрЃ«рЃЊрЃћрЃЉрЃљ.

## рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃцрЃљрЃЦрЃбрЃў
`<div>` рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃљ рЃЊрЃљ рЃБрЃЏрЃћрЃбрЃћрЃАрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃљрЃЊ. 

## рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃбрЃћрЃњрЃў
рЃбрЃћрЃњрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃЮрЃЉрЃўрЃА рЃюрЃўрЃерЃюрЃўрЃЌ `<` рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃћрЃбрЃЮрЃЉрЃўрЃА рЃюрЃўрЃерЃюрЃўрЃЌ `>` рЃљрЃарЃўрЃА рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃбрЃћрЃњрЃў рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃљ `<h1>`, `<p>` рЃЊрЃљ рЃљ.рЃе. 

## рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃў
рЃбрЃћрЃњрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЏрЃћрЃбрЃЮрЃЉрЃўрЃА рЃюрЃўрЃерЃюрЃўрЃЌрЃљ рЃЊрЃљ рЃЊрЃљрЃ«рЃарЃўрЃџрЃў рЃ«рЃљрЃќрЃўрЃЌ `</`, рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃЮрЃЉрЃўрЃА рЃюрЃўрЃерЃюрЃўрЃЌ `>` рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃў рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃљ `</h1>`,`</p>` рЃЊрЃљ рЃљ.рЃе.

## HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў
рЃарЃЮрЃњрЃЮрЃарЃф рЃќрЃћрЃЏрЃЮрЃЌ рЃЋрЃљрЃ«рЃАрЃћрЃюрЃћрЃЌ, HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃбрЃћрЃњрЃўрЃА, рЃЎрЃЮрЃюрЃбрЃћрЃбрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃўрЃА рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџрЃў рЃћрЃарЃЌрЃЮрЃЉрЃџрЃўрЃЮрЃЉрЃўрЃАрЃњрЃљрЃю. 

рЃљрЃАрЃћ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `<h1> рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў </h1>` рЃљрЃю `<p> рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў </p>` - рЃљрЃЏ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃерЃў рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃАрЃарЃБрЃџ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃўрЃгрЃДрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃбрЃћрЃњрЃўрЃЌ рЃЊрЃљ рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃўрЃЌ рЃ«рЃЮрЃџрЃЮ "рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў" рЃЊрЃљ "рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃў"  рЃљрЃарЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃбрЃюрЃў рЃарЃЮрЃЏрЃћрЃџрЃў рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃљрЃЏ рЃњрЃљрЃЏрЃ«рЃюрЃћрЃџ рЃЊрЃљ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋ рЃбрЃћрЃњрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА.

## HTML рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃў (headings)
HTML - рЃерЃў рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ 6 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃЌрЃљрЃф heading рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃЏрЃћрЃџрЃЮрЃЉрЃўрЃА/рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃЌ рЃЕрЃљрЃЋрЃгрЃћрЃарЃЌ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `h1` - рЃерЃў рЃўрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЉрЃљ "heading 1" рЃарЃљрЃф рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃўрЃЏрЃљрЃќрЃћ рЃарЃЮрЃЏ `h1` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў. 

рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃА рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЕрЃЋрЃћрЃюрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃЮрЃЏ рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЦрЃЋрЃћрЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃўрЃА рЃўрЃћрЃарЃљрЃЦрЃўрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ.

рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА `h1`, `h2`, `h3`, `h3`, `h5` рЃЊрЃљ `h6` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў - рЃарЃљрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃарЃЮрЃЏ 6 рЃЊрЃЮрЃюрЃћрЃќрЃћ - рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃбрЃћрЃЦрЃАрЃбрЃў рЃЊрЃљрЃЋрЃДрЃЮрЃЌ рЃАрЃћрЃЦрЃфрЃўрЃћрЃЉрЃљрЃЊ, рЃЦрЃЋрЃћ-рЃАрЃћрЃЦрЃфрЃўрЃћрЃЉрЃљрЃЊ рЃЊрЃљ рЃЏрЃљрЃЌ рЃАрЃљрЃЌрЃљрЃБрЃарЃћрЃЉрЃљрЃЊ. 

рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃћ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў:

```html
<h1>рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h1>
<h2>рЃЏрЃћрЃЮрЃарЃћ рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h2>
<h3>рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h3>
<h4>рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h4>
<h5>рЃЏрЃћрЃ«рЃБрЃЌрЃћ рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h5>
<h6>рЃЏрЃћрЃћрЃЦрЃЋрЃАрЃћ рЃЊрЃЮрЃюрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h6>
```

HTML - рЃерЃў рЃърЃљрЃарЃљрЃарЃљрЃцрЃўрЃА/рЃљрЃЉрЃќрЃљрЃфрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ `<p>` рЃбрЃћрЃњрЃў. 
рЃЏрЃњрЃљрЃџрЃўрЃЌрЃў:
```html
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
```

## HTML рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃў
рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ anchor рЃбрЃћрЃњрЃА рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃўрЃЌ `<a>` рЃбрЃћрЃњрЃўрЃЌрЃљрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў рЃЊрЃљ рЃАрЃарЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:

```html
<a href="https://www.example.com">This is a link</a>
```

**рЃљрЃЦ рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ?** рЃљрЃЦрЃљрЃЏрЃЊрЃћ рЃЋрЃЌрЃЦрЃЋрЃўрЃЌ рЃарЃЮрЃЏ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃбрЃћрЃњрЃўрЃА, рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃўрЃАрЃњрЃљрЃю рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ, рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА, рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџ рЃбрЃћрЃњрЃерЃў рЃЎрЃўрЃЊрЃћрЃЋ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ `href` рЃљрЃЦрЃЋрЃА рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў. 
**рЃћрЃА рЃарЃљ рЃљрЃарЃўрЃА?** рЃћрЃА HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃљ рЃЊрЃљ рЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃњрЃљрЃю рЃерЃћрЃњрЃЋрЃ«рЃЋрЃЊрЃћрЃЉрЃљ, рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃўрЃЌ. рЃљрЃЏ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў `href` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃўрЃЏ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃюрЃЊрЃљ рЃњрЃљрЃўрЃ«рЃАрЃюрЃљрЃА рЃЏрЃљрЃАрЃќрЃћ рЃЊрЃљрЃЎрЃџрЃўрЃЎрЃћрЃЉрЃўрЃАрЃљрЃА. 

рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЎрЃўрЃЊрЃћрЃЋ рЃБрЃцрЃарЃЮ рЃЋрЃарЃфрЃџрЃљрЃЊ рЃЋрЃўрЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃЌ [рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃЌрЃљрЃЋрЃерЃў](/courses/html/3-atributebi).

## HTML рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў
рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃерЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ `<img>` рЃбрЃћрЃњрЃћрЃЉрЃА. рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃАрЃћ рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ:

```html
<img src="image.jpg" alt="An example image">
```
рЃарЃЮрЃњрЃЮрЃарЃф рЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃљрЃЦрЃљрЃф рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃЮрЃЏ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃерЃћрЃЋрЃФрЃџрЃЮрЃЌ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `src` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў рЃЋрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃЌ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃА рЃЊрЃљ `alt` рЃбрЃћрЃњрЃерЃў рЃЋрЃгрЃћрЃарЃЌ рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА, рЃљрЃюрЃБ рЃўрЃАрЃћрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃЌрЃБ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃЋрЃћрЃа рЃ«рЃћрЃарЃ«рЃЊрЃћрЃЉрЃљ.

рЃЌрЃБ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ, `img` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃў рЃљрЃа рЃљрЃЦрЃЋрЃА рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃарЃљрЃўрЃЏрЃћ рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃА рЃљрЃа рЃерЃћрЃўрЃфрЃљрЃЋрЃА, рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљрЃф рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃўрЃљ рЃарЃЮрЃЏ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃљрЃЏ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЕрЃљрЃАрЃЏрЃљ/рЃЊрЃљрЃ«рЃљрЃбрЃЋрЃљ рЃерЃћрЃЋрЃФрЃџрЃЮрЃЌ. HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљрЃА рЃљрЃЦрЃЋрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃљрЃЏрЃ«рЃАрЃюрЃћрЃџрЃў рЃўрЃАрЃћ рЃЊрЃљрЃЏрЃ«рЃБрЃарЃљрЃЋрЃў рЃбрЃћрЃњрЃћрЃЉрЃўрЃф, рЃЌрЃБрЃЏрЃфрЃљ `img` рЃЊрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃЏрЃЮрЃюрЃљрЃЎрЃџрЃўрЃАрЃўрЃљ рЃЊрЃљ рЃЏрЃљрЃЌ рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃБрЃцрЃарЃЮ рЃЋрЃарЃфрЃљрЃџрЃЊ рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃЌрЃљрЃЋрЃћрЃЉрЃерЃў рЃњрЃљрЃЋрЃўрЃњрЃћрЃЉрЃЌ. 


## HTML рЃАрЃўрЃћрЃЉрЃў (lists)
HTML рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў рЃљрЃю рЃЊрЃљрЃБрЃџрЃљрЃњрЃћрЃЉрЃћрЃџрЃў рЃАрЃўрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ рЃЋрЃљрЃЕрЃЋрЃћрЃюрЃЮрЃЌ. 

рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃА рЃАрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃерЃў рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃф рЃарЃљрЃдрЃљрЃф рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃЌ рЃљрЃю рЃюрЃБрЃЏрЃћрЃарЃљрЃфрЃўрЃўрЃЌ рЃљрЃарЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў. рЃЏрЃљрЃА HTML - рЃерЃў Ordered List - рЃА рЃћрЃФрЃљрЃ«рЃўрЃљрЃю рЃЊрЃљ `<ol>` рЃбрЃћрЃњрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌрЃљрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў. 

рЃЊрЃљрЃБрЃџрЃљрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃўрЃАрЃћрЃЌрЃў рЃАрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃарЃљрЃўрЃЏрЃћ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃгрЃћрЃАрЃарЃўрЃњрЃўрЃЌ рЃљрЃа рЃљрЃарЃўрЃА рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў. рЃљрЃАрЃћрЃЌ рЃАрЃўрЃљрЃА HTML - рЃерЃў Unordered List - рЃА рЃБрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃю рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃбрЃћрЃњрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃЏ рЃЮрЃарЃў рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃљрЃЉрЃарЃћрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃЌрЃљрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў `<ul>`.

рЃЊрЃљрЃБрЃџрЃљрЃњрЃћрЃЉрЃћрЃџ рЃљрЃю рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџ рЃАрЃўрЃљрЃерЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ List Item рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃљрЃЉрЃарЃћрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃЌ рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў `<li>` рЃбрЃћрЃњрЃўрЃЌрЃљрЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў.

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃерЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў рЃўрЃАрЃћ рЃЊрЃљрЃБрЃџрЃљрЃњрЃћрЃЉрЃћрЃџрЃў рЃАрЃўрЃћрЃЉрЃў:

```html
<!-- Ordered List -->
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<!-- Unordered List -->
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
```

рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ рЃАрЃБрЃџ рЃћрЃА рЃљрЃарЃўрЃА. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЌрЃљрЃЋрЃерЃў рЃЎрЃўрЃЊрЃћрЃЋ рЃБрЃцрЃарЃЮ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ HTML рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃљрЃЌ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљрЃА. рЃљрЃ«рЃџрЃљ рЃЎрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃљ! ­Ъњф

## рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў
